# Smart-Sort AI Waste Segregation System

An automated, data-driven hardware optimization framework designed to integrate structural analytics and artificial intelligence into circular economy waste management systems. This repository contains structural simulation modules and log analysis frameworks for automated hardware classification.

## 📌 Project Overview
The Smart-Sort AI Bin Insert utilizes real-time structural data processing to eliminate manual sorting inefficiencies. By analyzing critical physical indicators—such as material density and structural rigidity—the embedded system executes real-time classification models to segregate waste streams dynamically into Recyclable, Organic, or General Waste categories.

## 📊 Analytical Methodology & Core Architecture
The system utilizes sequential classification thresholds derived from physical engineering constraints and cost-benefit frameworks:

* **Data Ingestion:** Real-time sensor metrics capture continuous variables for object density and rigidity structural profiles.
* **Optimization Framework:** Employs linear threshold modeling to minimize classification time while optimizing sorting purity boundaries.
* **System Metrics:** Tracks tracking time efficiency (sec) and structural categorization logs to maintain operational auditing capabilities.

## 📁 Repository Structure
* `analytics_model.py`: Core python simulation engine displaying the automated structural classification routing mechanism.
* `waste_data_logs.csv`: Simulated production log dataset mapping historical sensor array values against calculated categories.
